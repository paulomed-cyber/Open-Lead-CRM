# 🗺 Open Lead CRM Roadmap

This document outlines the planned development path for Open Lead CRM.

## Phase 1: MVP (Q2 2026) - *Status: In Progress*
Focus: Essential lead management and sales visualization.

- [x] Base Architecture (Next.js + Prisma + Tailwind)
- [x] Core Database Schema (Leads, Tasks, Interactions)
- [x] Dashboard with business metrics
- [x] Lead Management (List, Detail View)
- [x] Visual Kanban Pipeline
- [x] Basic Task Management
- [x] Marketing Attribution Support (UTMs, GCLID)
- [x] Seed data for quick start

## Phase 2: User Experience & Foundation (Q3 2026)
Focus: Multi-user support and advanced management features.

- [ ] User Authentication (NextAuth.js)
- [ ] Multi-workspace / Multi-tenant support
- [ ] Global Search functionality
- [ ] Lead Creation/Editing forms
- [ ] Advanced Filters and Data Export (CSV/Excel)
- [ ] Mobile responsive optimizations
- [ ] Custom Status and Lead Sources configuration

## Phase 3: Automation & Integrations (Q4 2026)
Focus: Connecting with external tools and automating workflows.

- [ ] Webhooks for Lead Capture (Typeform, WordPress, etc.)
- [ ] Google Calendar & Outlook integration
- [ ] WhatsApp Direct links and message templates
- [ ] Email notifications for tasks
- [ ] Basic Analytics and Reporting
- [ ] API for external integrations

## Phase 4: Intelligence & Growth (2027)
Focus: AI-driven insights and scale.

- [ ] AI Lead Summaries
- [ ] Smart Lead Scoring
- [ ] Recommended Next Actions based on history
- [ ] Document/Proposal management
- [ ] Advanced Sales Forecasting
- [ ] Multi-language support (i18n)

---

We are always looking for feedback on what to build next. Open an issue on GitHub to suggest a feature!
